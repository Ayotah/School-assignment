A repository (repo) is a storage space where your project files are kept.

  • Can be local (on your computer) or remote (on GitHub)
  • Tracks changes using version control (Git)
  • Allows collaboration with others

Commit

A commit is a snapshot of your changes in the repository.

push

Push sends your committed changes to a remote repository.

clone

Clone creates a copy of a remote repository on your local machine.
